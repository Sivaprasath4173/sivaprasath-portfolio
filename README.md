# Neon Glow Portfolio

A premium, modern portfolio website for Sivaprasath C, built with React, Tailwind CSS, and Vite.

## Features

- **Modern Aesthetic**: Dark mode with vibrant neon accents and glassmorphism.
- **Dynamic Animations**: Smooth scroll reveals, floating icons, and micro-interactions.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Project Showcase**: Visual grid of recent work with high-quality previews.
- **Skills Matrix**: Interactive skill bars and category tags.

## Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Vercel

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   bun install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
